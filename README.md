# AeroOps
Integrated Airport Ground Handling Management System built with Laravel and React.

## 📖 About

AeroOps is a web-based application designed to manage airport ground handling operations efficiently. This project is built as a personal portfolio while learning software engineering, database design, Laravel, JavaScript, React, Git, and GitHub.

## 🎯 Project Goals

- Learn Software Development Life Cycle (SDLC)
- Learn Database Design
- Build a Full Stack Web Application
- Learn Laravel Framework
- Learn JavaScript & React
- Practice Git & GitHub Workflow
- Build a Professional Portfolio Project
