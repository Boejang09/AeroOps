# AeroOps

Integrated Airport Ground Handling Management System built with Laravel.

## 📖 About

AeroOps is a web-based application designed to manage airport ground handling operations efficiently. This project is developed as a personal portfolio to learn Software Engineering, Database Design, Laravel, MySQL, Git, and Full Stack Web Development.

## 🎯 Project Goals

- Learn Software Development Life Cycle (SDLC)
- Learn Database Design
- Build a Full Stack Web Application
- Learn Laravel Framework
- Learn Authentication & Authorization
- Practice Git & GitHub Workflow
- Build a Professional Portfolio Project
